This plugin came about as a result of my desire to fully automate the creation of my program documentation.

First, I found, updated and enhanced a fork of the Markdownj library. This modified code is now my own BEWSoftware MDj Core Library. Then I developed a command-line program to use it: BEWSoftware MDj CLI.

The next logical step was to develop a Maven Plugin. The BEWSoftware MDj Maven Plugin is one possible implementation of that requirement.
